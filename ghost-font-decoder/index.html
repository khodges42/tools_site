Ghost Font Decoder

Drop in a Ghost Font GIF and this tries to pull the text back out. It runs in your browser. Nothing gets uploaded.

Choose a GIF.

What it does

It compares two frames and asks whether each small patch lines up better after shifting left or right. Then it repeats that across the GIF and averages the guesses.

This is still rough. Small frame gaps and small windows usually work best. The windows overlap now, so the final mask is less blocky. Higher analysis width keeps more detail, but it uses more memory and takes longer.
