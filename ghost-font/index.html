Ghost Font

Type a message, watch it emerge from opposing motion, then export the animation as a GIF.

How it works

Every dot looks identical in a still image. Most drift one way, while dots currently crossing the hidden letter shapes drift the opposite way. The message exists in temporal change rather than in any single frame, so pausing or inspecting frames individually reveals little more than noise. It is not encryption, but it is a neat example of information encoded in motion.
