Rules of startled air

The repeated action is indirect disturbance. A clap is not a command; it is a temporary weather event.

  • Tap/click near birds to make a clap wave. Birds turn away from it.
  • Soft claps nudge. Hard claps scatter, but can save a bird from a kite.
  • Keep the flock coherent without bunching it into predators.
  • Green thermals calm birds; red kites split the flock and take stragglers.
  • Space claps at the center. R restarts.
First thirty seconds: do almost nothing. Watch the flock’s grain. Clap beside it, not on it, and learn how panic travels from bird to bird.
A static browser game about collective behavior. No avatar route, no grid, no reference target, no craft score, no task queue. The loop is read the flock → introduce a disturbance → live with the emergent consequences.
