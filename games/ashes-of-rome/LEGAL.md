# Fan project release checklist

This file is a practical pre-release checklist, not a request for legal analysis during normal development.

Before public release:

- Verify the current official Dark Pack / fan-content program requirements.
- Add the currently required official logo or branding.
- Include the exact currently required attribution and disclaimer language.
- Confirm the project remains free and within current program requirements.
- Keep public repository content broadly PG-13: no overt sexual depictions, gratuitous gore, hateful caricature, or real historical suffering used as decorative supernatural lore.

During ordinary development, build and test the game. Revisit this checklist when preparing an actual public release or when a concrete release blocker appears.
