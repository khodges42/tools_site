Borrowed Teeth Library

Run a twelve-round occult lending desk. Buy strange books, shelve them for adjacency bonuses, and satisfy patrons before gossip, mold, and curses shut the place down.

Choose an arrival, then choose a shelf slot. Matching tags feed patrons; cursed books are powerful but raise gossip.
Shelf rules: matching tags adjacent orthogonally add +1 power. Opposite tags adjacent add gossip. Empty slots are safe but weak. Replacing a book refunds one coin.
Tags are concrete library problems: fang, moon, salt, lace, engine, saint, river, ash. The best shelf is not the shelf with the biggest numbers; it is the shelf that can answer weird patrons without letting rumors breed.
